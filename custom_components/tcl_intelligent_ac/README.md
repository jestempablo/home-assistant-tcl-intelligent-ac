# TCL Intelligent AC

Home Assistant custom integration for TCL air conditioners controlled by the Intelligent AC app.

This integration was built and tested with TCL XA71I devices in the United States / Other region. Other TCL models, OEM brands, and account regions may work if they use the same BroadLink/DNA profile, but they are not tested.

For full installation, HACS, setup, maintenance, and troubleshooting notes, see the repository `README.md`.
